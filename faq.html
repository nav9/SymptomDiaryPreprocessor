Frequently Asked Questions (FAQ)

The application accepts plain text files (`.txt`). It is designed to be flexible, but a common format would be one entry per line, starting with a date and time. For example: `2024-12-25 08:30, Breakfast: oatmeal, coffee, Feeling: good`. The data cleaning step will help you fix any inconsistencies.

Yes. At each step of the data refining process, you will have the option to download a file containing your progress. You can then re-upload this file at a later time to continue where you left off. The application will automatically detect which step the data belongs to.

You only need an internet connection once to either clone the repository or run the `install.sh` script which downloads all the necessary libraries. After that, the entire application can be run completely offline from your local machine by opening the `index.html` file in your browser.
